# HTML 化学教学动画

适用于**高中化学**课堂演示的 HTML 动画合集。每个动画放在对应教材章节的文件夹下，双击 HTML 文件即可在浏览器中运行，无需安装任何软件。

## 使用方式

1. 下载仓库
2. 用浏览器打开对应的 `.html` 文件
3. 点击按钮，观察变化

无需联网，无需服务器，无需安装依赖。

## 贡献指南

- 放在对应教材章节的文件夹下（如 `必修二/`、`选择性必修一/`）
- 复杂项目可拆分为 HTML + JS + CSS 多文件，用文件夹组织；简单项目单文件即可
- 路径全部用相对路径，双击 HTML 即可打开，无需服务器或构建工具
- 原生 HTML + CSS + JS 优先；复杂效果（3D、物理引擎等）可引入外部库，但应尽量避免，使用时优先 CDN 链接
- Canvas 动画为主，按钮驱动交互
- 包含完整的化学原理解释文字
- **功能准确性第一，视觉美观第二**（详见 [开发规范](docs/开发规范.md)）
