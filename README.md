# HTML 化学教学动画

适用于**高中化学课堂演示**的 HTML 动画合集。项目以单文件 HTML 动画为主，使用原生 HTML、CSS、JavaScript 和 Canvas 实现，打开文件即可运行，无需安装项目依赖或执行构建步骤。

## 项目现状

- 当前共收录 **9 个化学教学动画**。
- 以 Canvas 动画和按钮交互为主，支持播放、暂停、重置及进度拖动的页面可直接在浏览器中演示。
- 项目优先保证化学示意的清晰性与教学准确性，再兼顾视觉表现。
- 页面尽量保持自包含，便于课堂展示、离线打开和分发。

## 使用方式

1. 下载或克隆仓库。
2. 用浏览器直接打开对应的 `.html` 文件。
3. 根据页面中的按钮和进度条观察动画过程。

无需安装服务器或 npm 依赖。部分 3D 页面如引入 CDN 外部库，运行时需要网络连接；普通 Canvas 页面可直接使用 `file://` 打开。

## 开发约定

- 优先使用原生 HTML、CSS、JavaScript；简单动画保持单文件结构。实现复杂效果可引用外部库。
- 页面和资源使用相对路径，确保可以直接双击 HTML 文件运行。
- Canvas 动画应提供清晰的按钮、状态提示和必要的文字说明。
- 新增页面放在对应教材章节目录下。
- 避免引入不必要的构建工具和依赖；需要外部 CDN 时应在页面中明确说明。

## 贡献指南

欢迎补充新的高中化学演示动画。提交前请确认：

- 文件位于合适的教材章节目录中。
- 页面可直接打开并完成基本交互。
- 动画中的化学示意、文字说明和结构表达准确。
- 使用相对路径，不依赖本地绝对路径。
