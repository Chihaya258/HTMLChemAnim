# HTML 化学教学动画

适用于**高中化学**课堂演示的 HTML 动画合集。每个动画是一个独立的单文件 HTML，双击即可在浏览器中运行，无需安装任何软件。

## 使用方式

1. 下载仓库
2. 用浏览器打开对应的 `.html` 文件
3. 点击按钮，观察变化

无需联网，无需服务器，无需安装依赖。

## 补充说明

项目目前只有 1 个动画，库存很少。

各位化学老师如果有想要的演示场景，欢迎提需求（开 Issue）。熟悉前端的老师可以直接提交新的 HTML 文件（PR），保持"单文件、双击即用"的风格即可。

## 贡献指南

- 每个动画一个 `.html`，放在对应教材章节的文件夹下（如 `必修二/`、`选择性必修一/`）
- 零依赖，不引入任何外部库或框架
- Canvas 动画为主，按钮驱动交互
- 包含完整的化学原理解释文字
